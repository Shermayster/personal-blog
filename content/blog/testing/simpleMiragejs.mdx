Our app asks for items from server api. We creating a simple server that catch a request and returns a mock response.
Let's improve it a little bit.

## Get rid of baseUrl inside Server

we can use urlPrefix

```js
new Server({
  routes() {
    this.urlPrefix = "https://myserver"
    this.get(`/items`, () => {
      return new Response(200, {}, ["foo", "bar"])
    })
  },
})
```
